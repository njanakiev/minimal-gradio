# minimal-gradio

Minimal [Gradio](https://gradio.app/) example application.

# Usage

Prepare environment with:

```bash
python -m venv ./venv
source venv/bin/activate
pip install -r requrirements.txt
```

Run Gradio with:

```bash
python minimal/app.py
```

Build and run with docker:

```bash
docker-compose up --build
```
